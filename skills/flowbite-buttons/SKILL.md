# Flowbite Buttons

Use the Tailwind CSS button component inside forms, as links, and more with support for multiple colors, sizes, and variations

## Available Variants

- Default button
- Button pills
- Gradient monochrome
- Gradient duotone
- Gradient outline
- Colored shadows
- Social buttons
- Payment buttons
- Outline buttons
- Button sizes
- Outline button sizes
- Button sizes with icon
- Buttons with icon
- Button with label
- Icon buttons
- Loader
- Disabled

## Usage

For full documentation with all variants and code examples, read `references/buttons.md`.
