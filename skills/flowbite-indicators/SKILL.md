# Flowbite Indicators

Use the indicator component to show a number count, account status, or as a loading label positioned relative to the parent component coded with Tailwind CSS

## Available Variants

- Default indicator
- Legend indicator
- Indicator count
- Status indicator
- Badge indicator
- Stepper indicator
- Loading indicator
- Spinner indicator
- Indicator position
