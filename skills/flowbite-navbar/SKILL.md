# Flowbite Navbar

The navbar component can be used to show a list of navigation links positioned on the top side of your page based on multiple layouts, sizes, and dropdowns

**Requires JavaScript:** This component uses Flowbite's JS via data attributes for interactivity.

## Available Variants

- Default navbar
- Navbar with dropdown
- Multi-level dropdown
- Sticky navbar
- Navbar with submenu
- Navbar with search
- Navbar with CTA button
- Language dropdown
- User menu dropdown
- Mega menu navbar
- Solid background
- Hamburger menu
- More examples
- JavaScript behaviour
