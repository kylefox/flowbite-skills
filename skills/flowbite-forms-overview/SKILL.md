# Flowbite Forms

Use the Tailwind CSS form and input elements such as checkboxes, radios, textarea, text inputs to collect information from users with Flowbite

**Requires JavaScript:** This component uses Flowbite's JS via data attributes for interactivity.

## Available Variants

- Default form
- Floating labels
- Input Sizes
- Disabled inputs
- Shadow inputs
- Helper text
- Input element with icon
- Input element with addon
- Form validation
- Textarea
- Select input
- Checkbox
- Radio buttons
- File upload
- Toggle Switch
- More examples
