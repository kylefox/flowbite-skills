# Flowbite File Input

Get started with the file input component to let the user to upload one or more files from their device storage based on multiple styles and sizes

**Requires JavaScript:** This component uses Flowbite's JS via data attributes for interactivity.

## Available Variants

- File upload example
- Helper text
- Multiple files
- Sizes
- Dropzone
- Dropzone with button
