# Flowbite Accordion

Use the accordion component to show hidden information based on the collapse and expand state of the child elements using data attribute options

**Requires JavaScript:** This component uses Flowbite's JS via data attributes for interactivity.

## Available Variants

- Default accordion
- Separated cards
- Always open
- Color options
- Flush accordion
- Arrow style
- Nesting accordions
- JavaScript behaviour
