# Flowbite Modal

Use the modal component to show interactive dialogs and notifications to your website users available in multiple sizes, colors, and styles

**Requires JavaScript:** This component uses Flowbite's JS via data attributes for interactivity.

## Available Variants

- Default modal
- Static modal
- Pop-up modal
- Form element
- Modal with CRUD
- Modal with radio inputs
- Modal with timeline
- Modal with progress bar
- Crypto wallet
- Sizes
- Placement
- More examples
- JavaScript behaviour
