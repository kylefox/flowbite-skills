# Flowbite Charts

Use the chart and graph components from Flowbite built with Tailwind CSS and ApexCharts to choose from line, area, bar, column, pie, and radial charts and customize them using JavaScript

**Requires JavaScript:** This component uses Flowbite's JS via data attributes for interactivity.

## Available Variants

- Getting started
- Area chart
- Line chart
- Column chart
- Bar chart
- Pie chart
- Donut chart
- Radial chart
- Options
- JavaScript behaviour
- More examples
