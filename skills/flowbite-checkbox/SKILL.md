# Flowbite Checkbox

Get started with the checkbox component to allow the user to select one or more options in the form of a square box available in multiple sizes and colors

**Requires JavaScript:** This component uses Flowbite's JS via data attributes for interactivity.

## Available Variants

- Checkbox example
- Disabled state
- Checkbox link
- Helper text
- Bordered
- Bordered with description
- Bordered with icon
- Checkbox list group
- Horizontal list group
- Checkbox dropdown
- Inline layout
- Colors
- Advanced layout
