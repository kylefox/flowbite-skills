# Flowbite Tabs

Use these responsive tabs components to create a secondary navigational hierarchy for your website or toggle content inside a container

## Available Variants

- Default tabs
- Tabs with underline
- Tabs with icons
- Pills tabs
- Vertical tabs
- Full width tabs
- Tabs with icons
- Interactive tabs
- Active tab style
- JavaScript behaviour
