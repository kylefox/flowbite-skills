# Flowbite Skeleton

The skeleton component can be used as an alternative loading indicator to the spinner by mimicking the content that will be loaded such as text, images, or video

## Available Variants

- Default skeleton
- Image placeholder
- Video placeholder
- Text placeholder
- Card placeholder
- Widget placeholder
- List placeholder
- Testimonial placeholder
- Accessibility
