# Flowbite Jumbotron

Use the jumbotron component to show a marketing message to your users based on a headline and image inside of a card box based on Tailwind CSS

## Available Variants

- Default jumbotron
- Background image
- Featured video
- Authentication form
- Gradient background
- Jumbotron with cards
- More examples
