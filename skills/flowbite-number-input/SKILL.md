# Flowbite Number Input

Use the number input component to set a numeric value inside a form field based on multiple styles, variants, and layouts that can be used in product pages, forms, and more

**Requires JavaScript:** This component uses Flowbite's JS via data attributes for interactivity.

## Available Variants

- Default number input
- ZIP code input
- Phone number
- Control buttons
- Control buttons with icon
- Counter input
- Currency input
- Credit card input
- Pin code input
- Number input with slider
- Convert currency
- Advanced control buttons
- Min and max values
- JavaScript behaviour
