# Flowbite Device Mockups

Use the device mockups component to add content and screenshot previews of your application inside phone and tablet frames coded with Tailwind CSS and Flowbite

## Available Variants

- Default mockup
- iPhone 12 mockup (iOS)
- Google Pixel (Android)
- Tablet mockup
- Laptop mockup
- Desktop mockup
- Smartwatch mockup
- Mockup colors

For full documentation with all variants and code examples, read `references/device-mockup.md`.
