# Flowbite Sidebar

Use the sidebar component to show a list of menu items and multi-level menu items on either side of the page to navigate on your website

## Available Variants

- Default sidebar
- Multi-level menu
- Content separator
- CTA button
- Logo branding
- Sidebar with navbar
- Off-canvas sidebar
- More examples
- Dashboard Layout
