# Flowbite Table

Use the table component to show text, images, links, and other elements inside a structured set of data made up of rows and columns of table cells

## Available Variants

- Default table
- Highlight
- Table layout
- Table styles
- Overflow scrolling
- Table search
- Table filter
- Table pagination
- Checkbox selection
- Table with users
- Table with products
- Table with modal
- Table colors
- More examples
