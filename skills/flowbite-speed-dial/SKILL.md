# Flowbite Speed Dial

The speed dial component can be used as a quick way to show a list of action buttons to a user when hovering or clicking on the main trigger element.

**Requires JavaScript:** This component uses Flowbite's JS via data attributes for interactivity.

## Available Variants

- Default speed dial
- Square speed dial
- Text inside button
- Text outside button
- Dropdown menu
- Alternative menu
- Positioning
- Alignment
- Triggering
- JavaScript behaviour
