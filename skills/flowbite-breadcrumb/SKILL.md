# Flowbite Breadcrumbs

Show the location of the current page in a hierarchical structure using the Tailwind CSS breadcrumb components

## Available Variants

- Default breadcrumb
- Solid background
- Header breadcrumb
- Breadcrumb with dropdown
- Breadcrumb with button
- Breadcrumb with navigation
