# Flowbite Mega Menu

Use the mega menu component as a full-width dropdown inside the navbar to show a list of menu items based on multiple sizes, variants, and styles.

**Requires JavaScript:** This component uses Flowbite's JS via data attributes for interactivity.

## Available Variants

- Default mega menu
- Mega menu with icons
- Full width dropdown
- Full width with CTA
- Full width with image
- More examples
