# Flowbite Links

The link component can be used to set hyperlinks from one page to another or to an external website when clicking on an inline text item, button, or card

## Available Variants

- Default link
- Button
- Paragraph
- Icon link
- CTA link
- Card link
- Image link
- Card inner link
