# Flowbite Range Slider

Get started with the range component to receive a number from the user anywhere from 1 to 100 by sliding form control horizontally based on multiple options

**Requires JavaScript:** This component uses Flowbite's JS via data attributes for interactivity.

## Available Variants

- Range slider example
- Disabled state
- Min and max
- Steps
- Sizes
- Labels
