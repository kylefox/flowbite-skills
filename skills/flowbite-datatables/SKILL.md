# Flowbite Datatables

Use the datatable component to search, sort, filter, export and paginate table data of rows and columns for your web application coded with the utility classes from Tailwind CSS

**Requires JavaScript:** This component uses Flowbite's JS via data attributes for interactivity.

## Available Variants

- Getting started
- Default datatable
- Table search
- Filtering data
- Sorting data
- Table pagination
- Selecting rows
- Export files
