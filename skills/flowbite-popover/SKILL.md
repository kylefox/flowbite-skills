# Flowbite Popover

Use the popover component to show detailed information inside a pop-up box relative to the element that is being clicked or hovered based on multiple styles

**Requires JavaScript:** This component uses Flowbite's JS via data attributes for interactivity.

## Available Variants

- Default popover
- User profile
- Company profile
- Image popover
- Description popover
- Progress popover
- Password strength
- Placement
- Triggering
- Offset
- Animation
- Disable arrow
- JavaScript behaviour
