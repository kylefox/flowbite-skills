# Flowbite Toast

Push notifications to your users using the toast component and choose from multiple sizes, colors, styles, and positions

**Requires JavaScript:** This component uses Flowbite's JS via data attributes for interactivity.

## Available Variants

- Default toast
- Colors
- Simple toast
- Undo button
- Toast message
- Push notification
- Interactive toast
- Toast illustration
- Toast progress bar
- Toast danger alert
- Toast warning alert
- Positioning
- JavaScript behaviour
