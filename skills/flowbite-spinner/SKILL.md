# Flowbite Spinner

Use the spinner component as a loader indicator in your projects when fetching data based on an animated SVG using the utility classes from Tailwind CSS

## Available Variants

- Default spinner
- Colors
- Sizes
- Alignment
- Spinner with card
- Progress spinner
- Buttons
- Accessibillity
