# Flowbite Stepper

Use the stepper component to show the number of steps required to complete a form inside your application based on Tailwind CSS

## Available Variants

- Default stepper
- Progress stepper
- Detailed stepper
- Vertical stepper
- Breadcrumb stepper
- Timeline stepper
- Stepper with form
