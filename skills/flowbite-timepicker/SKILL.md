# Flowbite Timepicker

Use the timepicker component from Flowbite to select the time of the day in terms of hours, minutes and even seconds using an input selector based on Tailwind CSS

**Requires JavaScript:** This component uses Flowbite's JS via data attributes for interactivity.

## Available Variants

- Default timepicker
- Timepicker with icon
- Timepicker with dropdown
- Timepicker with select
- Timepicker range selector
- Timerange with dropdown
- Timerange picker with toggle
- Inline timepicker buttons
- Modal with timepicker
- Drawer with timepicker
