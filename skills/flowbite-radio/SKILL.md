# Flowbite Radio

Get started with the radio component to let the user choose a single option from multiple options in the form of a circle based on multiple styles and colors

**Requires JavaScript:** This component uses Flowbite's JS via data attributes for interactivity.

## Available Variants

- Radio example
- Disabled state
- Radio link
- Helper text
- Bordered
- Radio list group
- Horizontal list group
- Radio in dropdown
- Inline layout
- Advanced layout
- Advanced layout with icons
