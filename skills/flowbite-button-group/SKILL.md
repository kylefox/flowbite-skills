# Flowbite Button Group

Button groups are a Tailwind CSS powered set of buttons sticked together in a horizontal line

## Available Variants

- Default example
- Button group info
- Button group icon action
- Button group icons
- Button group dropdown
- Button group badge
- QR code button group
- Pagination button group
- Vertical button groups
- Button group with colors
- Button group as links
- Group buttons with icons
- Button group outline
