# Flowbite Blockquote

The blockquote component can be used to quote text content from an external source that can be used for testimonials, reviews, and quotes inside an article

## Available Variants

- Default blockquote
- Solid background
- Blockquote icon
- Paragraph context
- User testimonial
- User Review
- Alignment
- Sizes
