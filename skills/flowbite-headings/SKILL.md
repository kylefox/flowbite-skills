# Flowbite Headings

The heading component defines six levels of title elements from H1 to H6 that are used as titles and subtitles on a web page based on multiple styles and layouts

## Available Variants

- Default heading
- Second-level heading
- Highlighted heading
- Heading mark
- Heading gradient
- Heading underline
- Breadcrumb context
- Badge context
- Secondary text
- Sizes
