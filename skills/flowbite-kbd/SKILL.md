# Flowbite KBD (Keyboard)

Use the KBD component as an inline element to denote textual user input from the keyboard inside paragraphs, tables, and other components

## Available Variants

- Default KBD
- KBD inside text
- KBD inside table
- Arrow keys
- Letter keys
- Number keys
- Function keys
