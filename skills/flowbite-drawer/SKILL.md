# Flowbite Drawer (offcanvas)

The Drawer component can be used as a hidden off-canvas sidebar for navigation and to show other information based on multiple styles and placements

**Requires JavaScript:** This component uses Flowbite's JS via data attributes for interactivity.

## Available Variants

- Default drawer
- Drawer navigation
- Contact form
- Form elements
- Placement
- Body scrolling
- Backdrop
- Swipeable edge
- More examples
- JavaScript behaviour
