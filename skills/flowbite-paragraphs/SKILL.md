# Flowbite Paragraphs

Use the paragraph component to create multiple blocks of text separated by blank lines and write content based on multiple layouts and styles based on Tailwind CSS

## Available Variants

- Default paragraph
- Leading paragraph
- First letter
- Paragraph link
- Paragraph bold
- Paragraph underline
- Paragraph italic
- Paragraph popover
- Layout
- Text alignment
