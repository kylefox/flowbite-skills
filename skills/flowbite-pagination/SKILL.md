# Flowbite Pagination

Use the Tailwind CSS pagination element to indicate a series of content across various pages based on multiple styles and sizes

## Available Variants

- Default pagination
- Pagination with icons
- Previous and next
- Previous and next with icons
- Table data pagination
- Table pagination with icons
- Pagination with dropdown
- Pagination with input
- Input field and button
- Select input and buttons
- Single pagination
