# Flowbite Typography

Use the typography plugin from Flowbite to apply styles to all inline elements like headings, paragraphs, lists, and images using a single format class

## Available Variants

- Getting started
- Basic usage
- Link colors
- Dark mode
- Max width
- Disable format
- Options
- Blog templates
