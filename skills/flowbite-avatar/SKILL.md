# Flowbite Avatar

Use the avatar component to show a visual representation of a user profile using an image element or SVG object based on multiple styles and sizes

## Available Variants

- Default avatar
- Bordered
- Placeholder icon
- Placeholder initials
- Avatar tooltip
- Dot indicator
- Stacked
- Avatar text
- User dropdown
- Sizes
